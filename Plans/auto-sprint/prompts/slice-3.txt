#### Test 1: `AutoSprint_WalkRampsToSprintAfterDelay`

```
SetMoveInputForTest(Vector2.up)
SetSprintInputForTest — do NOT call this (let auto-sprint drive it)
```

- Warm up 30 frames (settled, not yet moving for long)
- Apply move input. Run 80 frames (~0.8s — before 1.2s threshold)
- Assert horizontal speed < walkSpeedCap * 1.2 (still at walk pace)
- Continue running to 220 frames total (~2.2s — well past threshold + blend)
- Assert horizontal speed > walkSpeedCap * 1.5 (clearly in sprint territory)

#### Test 2: `AutoSprint_StoppingResetsTimer`

- Warm up 30 frames
- Apply move input. Run 220 frames (reach sprint)
- Assert at sprint: speed > walkSpeedCap * 1.5
- Remove move input. Run 20 frames (0.2s — past 0.1s grace window)
- Reapply move input. Run 80 frames
- Assert back to walk pace: speed < walkSpeedCap * 1.2 (timer reset, walking again)

---

#### Test 3: `AutoSprint_LandIntoRun_MaintainsSpeed`

- Warm up 30 frames
- Ramp to sprint: SetMoveInputForTest(Vector2.up), run 220 frames
- Record pre-jump horizontal speed as `preJumpSpeed`
- SetJumpInputForTest(true), yield WaitForFixedUpdate, SetJumpInputForTest(false)
- Run up to 300 frames waiting for: first go airborne (IsGrounded == false), then land (IsGrounded == true)
- Once landed, run 15 more frames
- Assert horizontal speed > preJumpSpeed * 0.75
  (75% threshold — some physics deceleration is fine, a full stop is not)

---

#### Test 4: `AutoSprint_WalkJump_DoesNotLandAtSprintSpeed`

- Warm up 30 frames
- SetSprintInputForTest(false) — force walk (override)
- SetMoveInputForTest(Vector2.up), run 30 frames (walking, not sprinting)
- SetJumpInputForTest(true), yield WaitForFixedUpdate, SetJumpInputForTest(false)
- Wait up to 200 frames for land (same airborne→grounded pattern)
- Once landed, run 30 more frames
- Assert horizontal speed < walkSpeedCap * 1.3 (not at sprint speed)
