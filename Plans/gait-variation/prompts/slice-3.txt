## What this slice does
Adds a consistent per-leg stride asymmetry multiplier. The left leg takes a fractionally
longer stride than the right (4% by default). This is subconscious -- not a limp, just
a subtle personality. Serialized as prefab overrides so it can be tuned per character.

## Part A -- Implementation (FIRST commit)

File: Assets/Scripts/Character/LegAnimator.cs

### Step 1 -- Add serialized fields (near _noiseSeed and _disableOrganicVariation)
  [SerializeField, Tooltip("Stride length multiplier for the left leg. 1.0 = symmetric. " +
      "Values slightly above 1.0 give a subtle natural asymmetry (e.g. 1.04).")]
  private float _leftStrideAsymmetry = 1.0f;

  [SerializeField, Tooltip("Stride length multiplier for the right leg. 1.0 = symmetric.")]
  private float _rightStrideAsymmetry = 1.0f;

Default in code: 1.0f for both (symmetric). The prefab will override left to 1.04f.
Do NOT change the C# default -- prefab override only.

### Step 2 -- Apply in GetEffectiveStepAngle
In GetEffectiveStepAngle(float sprintNormalized, bool isLeftLeg), apply the asymmetry
BEFORE adding the per-stride noise. Order matters: asymmetry is a baseline shift,
noise is variation around that baseline.

  // Apply natural stride asymmetry first, then add per-stride noise around that baseline.
  float asymmetry = isLeftLeg ? _leftStrideAsymmetry : _rightStrideAsymmetry;
  if (_disableOrganicVariation) asymmetry = 1.0f;
  effectiveStepAngle *= asymmetry;

Then add noise and clamp as before. Final clamp [30, 90] still applies after all of this.
