### Feature 1: Per-Stride Amplitude Variation

Each arm gets a small random amplitude multiplier that changes once per stride cycle (when that arm crosses phase 0). This makes the two arms feel subtly independent — one arm might swing a little wider than the other on a given step.

**New serialized fields on ArmAnimator:**
```csharp
[Header("Organic Arm Variation")]
[SerializeField, Tooltip("Maximum amplitude variation per stride as a fraction of effective swing angle. " +
    "0.12 = ±12% variation. Bypassed when LegAnimator._disableOrganicVariation is true.")]
[Range(0f, 0.3f)]
private float _armAmplitudeVariation = 0.12f;
```

**New private state:**
```csharp
private System.Random _armRngLeft;   // seeded from LegAnimator seed + 5000
private System.Random _armRngRight;  // seeded from LegAnimator seed + 6000
private float _leftArmAmplitudeMod;  // current per-stride multiplier, range [1 - variation, 1 + variation]
private float _rightArmAmplitudeMod; // current per-stride multiplier
private bool _leftArmCrossed;        // true after crossing phase 0, false when re-triggered
private bool _rightArmCrossed;       // same for right
```

**Seeding:** In `Awake` (or wherever LegAnimator is accessed), initialise the RNGs:
```csharp
int seed = _legAnimator != null ? _legAnimator.GetOrganicSeedForTest() : 42;
_armRngLeft  = new System.Random(seed + 5000);
_armRngRight = new System.Random(seed + 6000);
_leftArmAmplitudeMod  = 1f;
_rightArmAmplitudeMod = 1f;
```

To get the seed from LegAnimator, add this public method to LegAnimator.cs:
```csharp
public int GetOrganicSeedForTest() => _noiseSeed;
```

**Per-stride update (in FixedUpdate, after reading legPhase):**

Left arm crosses phase 0 when `sinLeft` changes sign from negative to positive (arm starts forward swing). Right arm crosses when `sinRight` changes sign similarly. On each crossing, roll a new multiplier:

```csharp
bool sinLeftPositive  = sinLeft  >= 0f;
bool sinRightPositive = sinRight >= 0f;

if (!_disableArmVariation)
{
    // Left arm: regenerate on positive zero-crossing
    if (sinLeftPositive && !_leftArmCrossed)
    {
        _leftArmCrossed = true;
        double r = _armRngLeft.NextDouble(); // [0, 1)
        _leftArmAmplitudeMod = (float)(1.0 - _armAmplitudeVariation + r * 2.0 * _armAmplitudeVariation);
    }
    else if (!sinLeftPositive)
    {
        _leftArmCrossed = false;
    }

    // Right arm: same pattern
    if (sinRightPositive && !_rightArmCrossed)
    {
        _rightArmCrossed = true;
        double r = _armRngRight.NextDouble();
        _rightArmAmplitudeMod = (float)(1.0 - _armAmplitudeVariation + r * 2.0 * _armAmplitudeVariation);
    }
    else if (!sinRightPositive)
    {
        _rightArmCrossed = false;
    }
}
else
{
    _leftArmAmplitudeMod  = 1f;
    _rightArmAmplitudeMod = 1f;
}
```

**Apply the multiplier** after computing `leftSwingDeg` / `rightSwingDeg` but BEFORE the hard cap clamp:
```csharp
leftSwingDeg  *= _leftArmAmplitudeMod;
rightSwingDeg *= _rightArmAmplitudeMod;
```

**Gate:** `_disableArmVariation` — a private bool computed each frame:
```csharp
bool _disableArmVariation => _legAnimator == null || _legAnimator.IsOrganicVariationDisabled;
```

Add this public property to LegAnimator.cs:
```csharp
public bool IsOrganicVariationDisabled => _disableOrganicVariation;
```

### Feature 2: Left/Right Phase Offset

A small, fixed per-arm phase offset creates a subtle asymmetry in arm timing — the arms don't swing in perfect anti-phase with each other. This stacks with stride asymmetry from Slice 3 to make the gait feel genuinely personal rather than mechanical.

**New serialized field on ArmAnimator:**
```csharp
[SerializeField, Tooltip("Fixed phase offset (radians) added to the left arm's swing phase. " +
    "0.05 = ~3° of phase lead/lag vs the right arm. Bypassed when organic variation is disabled.")]
[Range(-0.2f, 0.2f)]
private float _leftArmPhaseOffset = 0.05f;
```

**Apply in FixedUpdate**, when computing sinLeft:
```csharp
float leftPhaseOffset = _disableArmVariation ? 0f : _leftArmPhaseOffset;
float sinLeft  = Mathf.Sin(legPhase + Mathf.PI + leftPhaseOffset);
float sinRight = Mathf.Sin(legPhase);  // right arm unchanged
```
