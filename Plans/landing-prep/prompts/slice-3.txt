## YOUR TASK: Slice 3 — Post-Landing Horizontal Velocity Damping

### New serialized fields on PlayerMovement:

```csharp
[Header("Jump Landing Damping")]
[SerializeField, Tooltip("Fraction of horizontal speed to PRESERVE each frame during the landing damping window. " +
    "0.75 = 25% bleed per application; spread across _jumpLandingHorizontalDampingDuration. " +
    "1.0 = no damping (disabled). Must be > 0.")]
private float _jumpLandingHorizontalDampingFactor = 0.75f;

[SerializeField, Tooltip("Seconds after touchdown during which horizontal damping applies. " +
    "Keeps damping brief so speed recovers quickly for locomotion.")]
private float _jumpLandingHorizontalDampingDuration = 0.08f;
```

### New private state:

```csharp
private float _jumpLandingDampingTimer;
```

### Implementation:

**On landing detection** (in the `if (!_jumpLandingDetected)` block at line 498, after setting `_jumpLandingDetected = true`):

```csharp
// Start horizontal damping window
if (_jumpLandingHorizontalDampingFactor < 0.9999f && _jumpLandingHorizontalDampingDuration > 0f)
{
    _jumpLandingDampingTimer = _jumpLandingHorizontalDampingDuration;
}
```

**In FixedUpdate, drive the damping** (add near the landing detection block, after it):

```csharp
if (_jumpLandingDampingTimer > 0f)
{
    _jumpLandingDampingTimer -= Time.fixedDeltaTime;

    // Compute per-frame factor so total effect equals _jumpLandingHorizontalDampingFactor
    // spread over _jumpLandingHorizontalDampingDuration.
    // At 100Hz over 0.08s = 8 frames. Per-frame retain = factor^(1/frameCount).
    int totalFrames = Mathf.Max(1, Mathf.RoundToInt(_jumpLandingHorizontalDampingDuration / Time.fixedDeltaTime));
    float perFrameRetain = Mathf.Pow(_jumpLandingHorizontalDampingFactor, 1f / totalFrames);

    Vector3 vel = _rb.linearVelocity;
    Vector3 horizontalVel = new Vector3(vel.x, 0f, vel.z);
    Vector3 dampedHoriz = horizontalVel * perFrameRetain;
    Vector3 delta = dampedHoriz - horizontalVel; // always <= 0 magnitude (reducing speed)
    _rb.AddForce(delta, ForceMode.VelocityChange);
}
```

**Cancel damping if entering a new jump wind-up** (wherever wind-up is entered in PlayerMovement):
```csharp
_jumpLandingDampingTimer = 0f;
```
